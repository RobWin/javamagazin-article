=== Currying

Currying erlaubt die partielle Anwendung einer Funktion durch das Umwandeln einer Funktion mit N Argumenten in eine modifizierte Funktion mit N-1 Argumenten. Mathematisch bedeutet dies beispielsweise, dass eine Funktion mit zwei Argumenten f: A1 x A2 -> B in eine modifizierte Funktion mit einem Argument fâ€™: A1 -> (A2 -> B) umgewandelt wird. Das festsetzen eines oder mehrerer Parameter erfolgt in Javaslang von links nach rechts. Siehe Listing 7.

[source,java]
.Listing 7
----
Function2<Integer, Integer, Integer> sum = (a, b) -> a + b;
// Parameter a wird mit 2 festgeschrieben
Function1<Integer, Integer> add2 = sum.curried().apply(2); 

then(add2.apply(4)).isEqualTo(6);
